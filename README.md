# Car-Rental-System
The proposed project, Car Rental System, is designed and implemented using Object-Oriented Programming (OOP) principles in Java. The system automates the core operations of a car rental service, such as managing cars, registering customers, and handling the renting and returning of cars.

This project is a simple Car Rental System built using Object-Oriented Programming (OOP) concepts in Java along with a Swing-based Graphical User Interface (GUI).
It allows users to view available cars, rent a car, and return a rented car. Each car is uniquely identified using a Car ID, making the rental and return process easy and efficient.
